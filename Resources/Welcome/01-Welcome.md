# Welcome to AI.md Reader

**AI.md Reader** is a simple app for reading markdown files. If you've ever received a `.md` file and wondered what to do with it — you're in the right place.

## What's a Markdown File?

Markdown is a way to format text using simple symbols:
- `#` makes a heading
- `**bold**` makes **bold** text
- `-` makes a bullet list (like this one)

Many AI tools (like ChatGPT, Claude, and others) write their responses in markdown. AI.md Reader helps you read these files beautifully.

## What Can You Do?

1. **Browse folders** — See the file tree on the left and tap folders to expand them
2. **Read beautifully** — Markdown is displayed with nice formatting right here
3. **Ask AI** — Toggle the AI chat panel to ask questions about any document

## Getting Started

You're already using the app! The sidebar on the left shows your folder structure. Click any markdown file to read it here.

When you're ready to explore your own files, close this window and use the launcher to open Desktop, Documents, Downloads, or any folder you choose.

### Credits

**Markdown Syntax Highlighting**  
This app's syntax highlighting is powered by custom NSRegularExpression patterns written specifically for markdown. No third-party parsing libraries were used — all highlighting code is original Swift using AppKit's NSTextView.

**Technologies Used**
- Swift & SwiftUI — Apple's modern frameworks
- AppKit NSTextView — Native text rendering
- Swift Concurrency — Actor-based isolation
- Apple Intelligence — On-device AI chat

**Privacy**  
AI.md Reader does not collect any data. All files stay on your device. Security-scoped bookmarks follow Apple's sandboxing guidelines.
