# Reading and Browsing

Pixley is a markdown reader first. Here's how to get the most out of it.

## Themes

Open **Settings** (`Cmd+,`) to pick a color theme: Xcode, GitHub, Solarized, Dracula, Monokai, Nord, or One Dark — each in light and dark variants.

## Fonts

Choose **System**, **Serif** (New York), **Sans-Serif** (SF Pro), or **Monospaced** (SF Mono). Adjust size with `Cmd++` / `Cmd+-` or the slider in Settings.

The heading scale controls how much larger headings are — compact, normal, or spacious.

## Sidebar

Folders show a file count. Click to expand/collapse (or use arrow keys). Type in the filter field to narrow by filename.

Right-click a file to star it. Toggle **Favorites Only** to see just starred files.

## Quick Switcher

`Cmd+P` opens a Spotlight-style search across every markdown file in your folder. Type, arrow, Enter.

## Opening Files

From the Start screen: **Open File** for a single `.md`, **Open Folder** to browse a tree, or drag onto the window. `Cmd+Shift+O` to open a new folder.

## File Watching

If something writes to the file you're reading — an AI tool, a build script — a reload banner appears. Click it or press `Cmd+R`.

## Bookmarks

Click any line number in the gutter to bookmark it. Bookmarks persist across sessions.

## Finding Text

`Cmd+F` to search, `Cmd+G` / `Cmd+Shift+G` for next/previous, Escape to close.
