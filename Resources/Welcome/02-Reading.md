# Reading Documents

Select a file from the sidebar and it appears here — headings, code blocks, lists, tables, and links are all styled by your current theme.

## Color Themes

Open **Settings** (`⌘,`) to pick a theme. Here's what's available:

- **Xcode** Light / Dark — Apple's default code colors
- **GitHub** Light / Dark — The look you know from GitHub
- **Solarized** Light / Dark — Ethan Schoonover's classic palette
- **Dracula** — Purple-accented dark theme
- **Monokai** — Sublime Text's iconic colors
- **Nord** — Arctic-inspired cool tones
- **One Dark** — Atom editor's popular dark scheme

## Fonts and Sizing

Choose between **System**, **Serif** (New York), **Sans-Serif** (SF Pro), or **Monospaced** (SF Mono). Adjust size with the slider in Settings or use `⌘+` and `⌘-` while reading.

The **heading scale** controls how much larger headings are relative to body text — compact, normal, or spacious.

## Line Bookmarks

Click any line number in the gutter to place a bookmark — an orange dot marks the spot. Click again to remove it. Bookmarks are saved across sessions.

Toggle the gutter off in Settings if you prefer a cleaner view.

## Finding Text

Press `⌘F` to open the find bar. Matches highlight as you type. `⌘G` for next, `⌘⇧G` for previous, Escape to close.

## File Watching

If something writes to the file you're reading — an AI tool, a build script, anything — a reload banner appears at the bottom. Click it or press `⌘R` to refresh.

## Reading Progress

A badge in the top-right shows how far you've scrolled. Your position is saved per file, even across restarts.
