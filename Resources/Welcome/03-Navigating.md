# Navigating Files

## Opening a Folder

From the Start screen, click **Open Folder** or use the shortcut buttons for Desktop, Documents, and Downloads. You can also drag a folder or a single `.md` file onto the window.

To switch folders later: `⌘⇧O` for the folder picker, or `⌘W` to close and return to Start.

## The Sidebar

Folders show an icon and a count of the markdown files inside (including subfolders). Click to expand or collapse. Arrow keys work too — Right to expand, Left to collapse.

## Filtering

Type in the filter field at the top of the sidebar to narrow by filename. Only matching files and their parent folders appear.

## Favorites

Right-click a file to star it. Toggle **Favorites Only** to see just your starred files — useful when a folder has dozens of files and you only care about a few.

## Quick Switcher

Press `⌘P` for a Spotlight-style search across every markdown file in your folder. Start typing, arrow to your result, Enter to jump.

```
⌘P → type "read" → ↓ → Enter
```

It matches filenames, so short queries work well.

## Recent Files

The Start screen shows your recently opened folders and files. Everything is one click away.
