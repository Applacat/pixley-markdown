# Keyboard Shortcuts

AI.md Reader supports these keyboard shortcuts:

## In the Browser

| Shortcut | Action |
|----------|--------|
| **⌘ R** | Reload the current document |
| **⌘ ⇧ O** | Open/change folder |

## In AI Chat

| Shortcut | Action |
|----------|--------|
| **ESC** | Forget conversation (clear chat) |
| **Enter** | Send your message |

## Navigation

| Shortcut | Action |
|----------|--------|
| **↑ / ↓** | Navigate files in sidebar |
| **Enter** | Open selected file |

---

*Tip: Most actions also have menu items you can discover in the menu bar.*
