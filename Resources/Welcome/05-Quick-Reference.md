# Quick Reference

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Open file | `Cmd+O` |
| Open folder | `Cmd+Shift+O` |
| Close folder | `Cmd+W` |
| Quick Switcher | `Cmd+P` |
| Find in document | `Cmd+F` |
| Next / previous match | `Cmd+G` / `Cmd+Shift+G` |
| Reload document | `Cmd+R` |
| Font size | `Cmd++` / `Cmd+-` |
| Toggle AI Chat | `Cmd+Shift+A` |
| Add Comment | `Cmd+Shift+C` |
| Next / previous element | `Tab` / `Shift+Tab` |
| Activate element | `Return` or `Space` |
| Settings | `Cmd+,` |

## Interactive Controls

| Markdown | Control | Result |
|----------|---------|--------|
| `- [ ]` / `- [x]` | Checkbox | Toggled `[ ]` or `[x]` |
| `> - [ ] Option` | Radio (single select) | `[x]` on chosen |
| `[[placeholder]]` | Text input | `[[entered value]]` |
| `[[pick date]]` | Date picker | `[[2026-03-27]]` |
| `[[choose file]]` | File dialog | `[[/path/to/file]]` |
| `[[choose folder]]` | Folder dialog | `[[/path/to/folder]]` |
| `<!-- status: a / b / c -->` | Status dropdown | `**Status:** current` |
| `<!-- feedback -->` | Gutter comment | `<!-- feedback: text -->` |
| `> - [ ] PASS / FAIL / ...` | Review with date | `[x] PASS - 2026-03-27` |
| `{++text++}` | Accept/Reject addition | `text` or removed |
| `{--text--}` | Accept/Reject deletion | removed or `text` |
| `{~~old~>new~~}` | Accept/Reject substitution | `new` or `old` |
| `{==text==}{>>note<<}` | Inline comment | Click highlight to read |

## Privacy

All AI processing happens on-device via Apple Intelligence. No data leaves your Mac.
