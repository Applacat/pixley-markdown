# Browsing Folders

The left sidebar shows all the files in your selected folder.

## How It Works

- **Folders** have a blue folder icon and show a number — that's how many markdown files are inside
- **Click a folder** to expand it and see what's inside
- **Click again** to collapse it
- **Markdown files** have a document icon — click to view them

## The Numbers Add Up

If a folder shows "13", that means there are 13 markdown files inside it (including subfolders). The numbers always add up correctly from children to parents.

## Scrolling

If file names are long, you can scroll horizontally to see the full name. The entire row is clickable, not just the text.

## Changing Folders

To open a different folder, simply close this window. The launcher will appear where you can pick a new folder.

You can also use **⌘ ⇧ O** (Command-Shift-O) to open a folder picker.

---

*Tip: You can drag and drop a folder onto the window to open it.*
