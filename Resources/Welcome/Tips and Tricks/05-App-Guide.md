# App Guide

## Themes

Open **Settings** (`⌘,`) to pick a color theme: Xcode, GitHub, Solarized, Dracula, Monokai, Nord, or One Dark — each in light and dark variants.

## Fonts

Choose **System**, **Serif** (New York), **Sans-Serif** (SF Pro), or **Monospaced** (SF Mono). Adjust size with `⌘+` / `⌘-` or the slider in Settings.

The heading scale controls how much larger headings are — compact, normal, or spacious.

## Sidebar

Folders show an icon and a count of markdown files inside. Click to expand/collapse (or use arrow keys). Type in the filter field to narrow by filename.

Right-click a file to star it. Toggle **Favorites Only** to see just your starred files.

## Quick Switcher

`⌘P` opens a Spotlight-style search across every markdown file in your folder. Type, arrow, Enter.

## Opening Folders

From the Start screen: **Open Folder**, shortcut buttons for Desktop/Documents/Downloads, or drag a folder onto the window. `⌘⇧O` to switch folders, `⌘W` to close.

## File Watching

If something writes to the file you're reading — an AI tool, a build script — a reload banner appears. Click it or press `⌘R`.

## Bookmarks

Click any line number in the gutter to bookmark it. Bookmarks persist across sessions.

## Finding Text

`⌘F` to search, `⌘G` / `⌘⇧G` for next/previous, Escape to close.
