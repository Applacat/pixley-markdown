## Implementation Notes

### Spec Deviations
- US-1.2/1.3: Appearance and Rendering merged into one settings tab (all controls present, cleaner UX)
- US-2.2/2.3: Content search and search navigation are one feature via native NSTextView find bar (usesFindBar + isIncrementalSearchingEnabled)
- Themes wired via SyntaxTheme/SyntaxPalette, not SwiftUITheme/MarkdownTheme protocol directly

### Key Architectural Decisions
1. AppCoordinator pattern for state management
2. aimdRenderer as local Swift Package
3. Themes = Renderers (pluggable output strategies)
4. SwiftData for persistence with FileMetadataRepository protocol
5. Apple swift-markdown for AST parsing
6. Split View deferred to applacatCanvas (v2.0/v3.0)
7. Phase 1.5/1.6 inserted for audit fixes (2026-02-04)
8. Async document coordination via callback on AppState
9. Error display via status bar (hidden normally, slides up on error)
