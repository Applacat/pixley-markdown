# Getting Started with AI.md Reader

Welcome! This guide will help you get the most out of AI.md Reader.

---

## Opening Folders

AI.md Reader works with folders containing markdown files. Here's how to open them:

### Method 1: Start Screen Shortcuts
When you launch the app, you'll see quick shortcuts:
- **Desktop** — Browse markdown files on your Desktop
- **Documents** — Open your Documents folder
- **Downloads** — Check your Downloads folder
- **Choose Folder...** — Pick any folder on your Mac

### Method 2: Drag & Drop
You can drag a folder from Finder directly onto the app window!

### Method 3: Menu Bar
Use **File → Choose Folder...** (⌘⇧O) to browse for any folder.

---

## Navigating Files

### Sidebar Navigation
The left sidebar shows your folder structure:
- **📁 Blue folders** — Click to expand/collapse
- **📄 Files** — Click to open and read
- **Number badges** — Shows how many markdown files are in each folder

### Keyboard Shortcuts
- **⌘R** — Reload current file (if it changed on disk)
- **⌘⇧O** — Change folder
- **⌘/** — Open this Help guide

---

## Reading Markdown

### Syntax Highlighting

AI.md Reader makes markdown beautiful with automatic styling:

- **# Headings** → Large, blue, bold text
- **## Subheadings** → Medium, indigo text
- **### Small headings** → Smaller, purple text
- **`code`** → Orange text with gray background
- **\*\*bold\*\*** → Bold text
- **\*italic\*** → Italic text
- **[links](url)** → Teal colored links
- **- Lists** → Gray bullet points
- **> Quotes** → Secondary color text
- **---** → Separator lines

### Font Size Controls

Use the toolbar buttons to adjust text size:
- **[−]** button → Decrease font size
- **[+]** button → Increase font size

Font size adjusts from **10pt to 32pt**.

---

## Performance Tips

### Large Files
AI.md Reader handles files up to **10 MB** in size. Files over **1 MB** will show as plain text (no syntax highlighting) for better performance.

### Many Files
The app caches folder structures for faster loading. If you add/remove files, use **⌘R** to reload.

### Smooth Typing
If you edit files in AI.md Reader, the highlighting is debounced (waits 150ms after you stop typing) to keep things smooth.

## Common Questions

### Q: Can I edit markdown files?
**A:** Yes! The text view supports editing. However, changes are not automatically saved. Use your favorite text editor for serious markdown editing.

### Q: What file types are supported?
**A:** Files with `.md` or `.markdown` extensions.

### Q: Why can't I see some files?
**A:** The sidebar only shows markdown files and folders containing markdown files. Other file types are hidden.

### Q: How do I switch folders?
**A:** Use **File → Change Folder...** (⌘⇧O) or drag a new folder onto the app.

### Q: Where are my recent folders?
**A:** The start screen shows recently opened folders and files. You can also reopen the last folder you had open.

## Advanced Features

### Session Restore
AI.md Reader remembers the last folder you had open and automatically reopens it on launch.

### First Launch Welcome
On first launch, the app shows this Welcome tutorial automatically. You can reopen it anytime from **Help → AI.md Reader Help**.

### Security-Scoped Bookmarks
The app uses macOS security features to remember folders you've granted access to, so you don't have to grant permission every time.

## Troubleshooting

### File Won't Open
- Check that it's a `.md` or `.markdown` file
- Ensure the file is under 10 MB
- Try **⌘R** to reload

### Folder Permission Denied
- The app needs permission to access folders
- When prompted, click "OK" to grant access
- For system folders (Desktop, Documents), you'll see a file picker

### Performance Issues
- Files over 1 MB won't have syntax highlighting (by design)
- Close other apps if your Mac is running slowly
- Try a smaller folder with fewer files

### Welcome Tutorial Disappeared
- Use **Help → AI.md Reader Help** to reopen it
- Or **Help → Getting Started** to see this guide

## Tips & Tricks

### 🎨 Organize Your Markdown
Create folders for different projects:
```
Documents/
  ├─ ProjectA/
  │  ├─ notes.md
  │  └─ ideas.md
  └─ ProjectB/
     ├─ readme.md
     └─ docs.md
```

### ⚡ Quick Access
Keep frequently-used markdown folders in your:
- Desktop (easy drag & drop)
- Documents/Markdown folder
- Downloads (for files from AI tools)

### 🔍 Find Files Fast
The sidebar shows folder badges with markdown counts. Look for high numbers to find documentation-heavy folders!

### 🌓 Dark Mode
AI.md Reader respects your system dark mode setting. Toggle it in **System Settings → Appearance**.
